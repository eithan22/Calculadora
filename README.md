# Calculadora 🔢

A **responsive web calculator** built with **HTML5, CSS3, and JavaScript**. Supports basic arithmetic operations with a clean, modern interface inspired by real calculator layouts.

---

## 📋 Description

This project is a fully functional browser-based calculator that handles addition, subtraction, multiplication, and division. It features a clear display, backspace functionality, and decimal support — all implemented with vanilla JavaScript DOM manipulation.

---

## ✨ Features

- ➕ **Basic Arithmetic** — Addition, subtraction, multiplication, division
- ⌫ **Backspace** — Delete last digit without clearing the full display
- 🔄 **Clear** — Reset display to zero with the C button
- 🔢 **Decimal Support** — Enter decimal numbers with the . button
- 📱 **Responsive Layout** — Adapts to different screen sizes via CSS Grid
